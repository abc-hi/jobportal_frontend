**Job Portal - Frontend**

This is the frontend of a Job Portal project. The focus of this repository is to provide an intuitive and responsive user interface for job seekers and employers.

**Technologies Used**

React.js – For building interactive UI components

React Router DOM – For client-side routing

Axios – For making API calls to the backend

Bootstrap – For styling and responsive design

CSS  – For custom styles

React Context API – For state management (user login status, filters, etc.)



**Features**

Home page with a banner and welcome message

Job listing with filter options

View job details on the right panel

Apply for jobs (redirects to login if not authenticated)

Save jobs (requires login)

Share job links on social media (LinkedIn / Twitter)

Related jobs displayed based on job category or skills
